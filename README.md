# 🛡️ SOC Home Lab

## 📌 Overview

This repository documents the development of a personal Security Operations Center (SOC) lab designed to simulate real-world enterprise security monitoring and incident response.

The objective is to gain practical Blue Team experience through hands-on detection, investigation, and threat analysis.

---

## 🎯 Objectives

- Build a multi-machine SOC lab
- Configure Wazuh SIEM
- Monitor Windows and Linux endpoints
- Analyze security logs
- Investigate alerts
- Perform threat hunting
- Simulate cyber attacks safely
- Map detections to the MITRE ATT&CK framework

---

## 🛠️ Technologies

- Wazuh
- Windows 11
